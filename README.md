# ExamBot

Голосовой ассистент на FastAPI, Whisper и Silero.

## Запуск в Docker

### Вариант для разработки (hot reload)

Используйте основной compose-файл и dev-оверлей, чтобы контейнер видел актуальный код и автоматически перезапускал Uvicorn при изменениях:

```bash
./run_dev.sh
```

- директория проекта монтируется в контейнер `/app`
- `uvicorn` стартует с флагом `--reload`
- Ollama поднимается соседним контейнером; первый запуск подтянет нужную модель (`docker compose exec ollama ollama pull llama3.1`)

### Вариант для тестового/production запуска

```bash
docker compose up --build
```

- исходники копируются в образ во время сборки
- контейнер запускается без автоматического перезапуска при изменениях кода

## HTTPS и мобильный доступ

Браузеры на iOS/Android требуют защищённый источник для доступа к микрофону. Для быстрого HTTPS можно использовать ngrok:

```bash
ngrok http http://localhost:8000
```

После запуска берите ссылку `https://...ngrok.app` из вывода и открывайте её на телефоне. Не забудьте предварительно указать authtoken (`ngrok config add-authtoken <TOKEN>`).
